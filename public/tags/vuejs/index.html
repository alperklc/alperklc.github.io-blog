A starter graphql project - Part 2 (UI with VueJs)

In March, I’ve developed and shared a starter graphql project, which is initially intended to be a bootstrapper for Nodejs based Graphql APIs. This time, I’ve added a vue-js frontend to the stack. It uses apollo-clientfor communicating with the backend. UI is optimized for mobile devices. Frontend application is capable to create an user login logout list existing records search records create a record edit a record delete a record Screenshots Live demo is available: https://vue-nodejs-graphql-starter.
